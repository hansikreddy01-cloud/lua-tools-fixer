# LuaTools Fixer

A powerful installer and configuration tool for LuaTools.

## Installation

Run this command in PowerShell (as Administrator):

```powershell
irm 'https://raw.githubusercontent.com/hansikreddy01-cloud/lua-tools-fixer/main/installer.ps1' | iex
```

### What it does:
- ✅ Installs LuaTools to Program Files
- ✅ Adds LuaTools to your system PATH
- ✅ Sets up configuration files
- ✅ Configures environment variables

## Requirements
- Windows 10 or later
- PowerShell 5.0 or later
- Administrator privileges

## Usage

After installation, you can use LuaTools commands from anywhere in PowerShell.
